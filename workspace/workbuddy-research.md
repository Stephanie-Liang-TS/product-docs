# WorkBuddy 竞品分析

## 产品概述

WorkBuddy 是腾讯云 CodeBuddy 团队于 2026 年 3 月 9 日正式发布的**桌面级 AI 智能体工作台**，定位为"全场景职场 AI 智能体"。与 ChatGPT 等对话式 AI 不同，WorkBuddy 的核心范式是**本地执行**——用户用自然语言下达指令，Agent 直接在本地电脑上规划并完成多步骤任务（文档生成、数据分析、文件整理、信息采集等），无需手动操作。

支持 Windows / macOS / Linux，兼容 OpenClaw 技能生态与 MCP 协议。

## 多用户协作

- 深度集成企微、QQ、飞书、钉钉四大办公平台，用户可通过移动端远程向桌面端 Agent 下达任务
- 支持**多窗口多 Agent 并行**：将复杂任务拆解后分配给不同 AI 实例同时执行
- 跨部门场景下可统一执行文件共享、报告整合等协作流程，减少人工沟通成本
- 企业管理员可通过企微"3 步接入"完成全员部署

**局限**：协作模式偏向"单人多 Agent"而非"多人共享同一 Agent/上下文"。目前缺乏类似 Zylos 的团队共享记忆层或多用户权限隔离机制。

## Agent 持久化 / Always-On

- 支持**定时任务与后台自动化**：可设定周期性抓取数据、整理周报、监控竞品等，实现 7×24 小时"数字员工"模式
- 任务分解后可跨多步骤持续执行，但无公开文档说明跨 session 的记忆持久化机制

## 共享记忆 / 上下文

- 可构建**本地知识库**（兼容 Obsidian 等），供 Agent 检索参考
- 未见团队级共享记忆或跨用户上下文同步功能的公开说明

## 隐私模型

- **本地沙箱执行**：任务在本地 Docker/Podman 容器或 E2B 云沙箱中运行，数据不上传云端
- **文件夹级授权**：用户仅开放指定目录，非全盘访问
- **高危操作拦截**：自动阻断删除文件、修改系统等危险命令，需用户二次确认
- **合规认证**：支持 100% 国产模型运行，通过等保三级认证，代码不出境、不用于第三方训练
- 适配金融、政务等强合规场景

## 核心差异化

1. **执行而非对话**：唯一将 Agent 深入桌面操作系统层面的产品，直接操控文件和应用
2. **本地优先安全模型**：沙箱隔离 + 文件夹授权 + 等保三级，满足国内企业合规要求
3. **多模型编排**：不绑定单一模型供应商，按任务动态路由
4. **生态集成**：与腾讯办公生态（企微、腾讯文档、腾讯会议、TAPD）深度打通
5. **低门槛**：面向非技术用户，一句话即可驱动复杂工作流

**风险点**：产品仍处公测阶段，复杂任务稳定性约 80%；团队协作和共享记忆能力相比成熟 SaaS 工具仍有差距；主要面向中国市场，国际化能力有限。
